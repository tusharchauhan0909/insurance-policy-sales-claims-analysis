# Car Insurance Portfolio Risk & Profitability Analysis

This project simulates and analyzes a large-scale car insurance portfolio containing approximately 1 million policies.

## Project Objective

The goal of this project is to evaluate:

- Premium revenue
- Claim patterns
- Policy profitability
- Portfolio loss ratio
- Future claim risk exposure

## Tools Used

Python (Jupyter Notebook)  
SQL (MySQL)  
Power BI  

## Project Workflow

1. Data Simulation using Python
2. Data Validation in Jupyter Notebook
3. Data Analysis using SQL
4. Business Insights Generation
5. Dashboard Visualization in Power BI

## Key Insights

- Total premium revenue generated: ₹24.01 Crores
- Total claim cost: ₹98.81 Crores
- Portfolio loss ratio: 4.11
- Most profitable policy tenure: 3-year policies
- Estimated future claim liability: ₹950.85 Crores
